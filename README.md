# Power BI Sales Analysis Dashboard

## Overview

This project is a comprehensive **Sales Analysis Dashboard** built using Power BI. It focuses on analyzing sales performance, customer behavior, delivery efficiency, and regional trends using interactive visualizations.

The dashboard provides actionable insights to support business decision-making and performance evaluation.

## Objectives

* Identify top-performing product categories
* Analyze delayed vs on-time deliveries
* Understand customer payment preferences
* Evaluate product ratings (top & bottom categories)
* Examine regional sales performance
* Detect seasonal trends in sales
* Analyze revenue growth over time

## Key Insights

* Health & Beauty category recorded the highest sales, while garden tools ranked lowest
* Bed, Bath & Table category had the highest number of delayed orders
* Sales peak during **Quarter 2**, showing seasonal trends
* Credit cards are the most preferred payment method
* São Paulo shows the highest sales, while Rio Branco has the lowest
* Revenue consistently increased from 2016 to 2018 

## Tools Used

* Power BI
* Data Visualization
* Business Intelligence Techniques

## Features

* Interactive dashboards
* Drill-through analysis for monthly delivery performance
* Dynamic filtering and cross-report insights
* Visual comparison of delayed vs early deliveries

## How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Interact with filters and visuals
3. Use drill-through to explore deeper insights
